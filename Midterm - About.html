About This Exam

This exam serves as a test on what we have learned this past half-semester.
Some aspects we as students must have mastered by this point includes:

  1. HTML: mastering typing in Markup Language format
  2. Anchoring: making hyperlink between pages
  3. Lists: making ordered and unordered lists in different formats
  4. Image embedding: adding images into a webpage with alt text
  5. CSS styling: using Cascading Style Sheet to customize the look of websites

We hope most of our work is qualified for the highest grade.
